# Womo UI

Este folder queda reservado para módulos visuales de Womo.

En esta primera fase se dejó la lógica visual dentro de `src/core/womo-core.js` para evitar romper dependencias globales. A partir de esta base, los nuevos componentes visuales pueden ir saliendo gradualmente aquí: home, hero, carruseles, modal preview, navegación y player-ui.
