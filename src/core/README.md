# Womo Core

Aquí vive el motor funcional de Womo Web.

No modificar este archivo para cambios puramente visuales. La versión actual conserva el comportamiento aprobado de:

- Firebase Auth
- Catálogo Firestore
- Favoritos
- Continuar viendo
- Progreso por usuario
- Player HLS
- Shuffle
- Overlay de siguiente episodio
- Protección contra eventos `ended` falsos al iniciar reproducción

Archivo principal: `womo-core.js`.
